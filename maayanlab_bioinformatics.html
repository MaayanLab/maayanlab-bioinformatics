Module contents¶

A collection of useful functions for bioinformatics data analysis.

This library contains many functions and methods I use again and again in different analyses including:

  • api: enrichr, l1000fwd, …

  • clustering: silhouette analysis, …

  • dge: various differential gene expression techniques including chdir, lima_voom, ..

  • harmonization: transcript to gene conversions, …

  • normalization: log, cpm, zscore, filterByExpr, quantile, …

  • parse: gmt, suerat ready files, …

  • utils: fetch_save_read, merge, …
